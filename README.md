This is a frontend clone of the Udemy website built using HTML and CSS. It replicates the user interface and layout of Udemy, focusing on responsive design, UI consistency, and web layout practices. This project was created for educational purposes to improve HTML and CSS skills.

✨ Features
✅ Responsive layout (desktop & mobile friendly)

✅ Homepage with banner, categories, and featured courses

✅ Course cards with pricing, ratings, and images

✅ Header with navigation and search

✅ Footer with links and social icons
